# Cleio Legal

Cleio iOS 앱의 이용약관 및 개인정보처리방침을 호스팅하는 GitHub Pages 사이트입니다.

## 사이트

https://cleioapp.github.io/

## 문서 목록

| 문서 | 한국어 | English |
|---|---|---|
| 개인정보처리방침 | [privacy-policy](https://cleioapp.github.io/privacy-policy/) | [privacy-policy-en](https://cleioapp.github.io/privacy-policy-en/) |
| 이용약관 | [terms-of-service](https://cleioapp.github.io/terms-of-service/) | [terms-of-service-en](https://cleioapp.github.io/terms-of-service-en/) |

## 정본

한국어 버전이 정본이며, 번역본과 충돌 시 한국어 정본이 우선합니다.

## 수정 절차

1. 본 리포의 마크다운 파일 수정
2. `main` 브랜치에 푸시
3. GitHub Pages 자동 빌드 (약 1분 소요)
4. 본 Cleio 앱 리포 `docs/legal/` 동기화 (정본 일치 유지)

## 빌드

GitHub Pages가 Jekyll로 자동 빌드합니다. 로컬 미리보기:

```bash
bundle install
bundle exec jekyll serve
```
